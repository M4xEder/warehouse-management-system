
# 📦 Sistema de Gerenciamento de Armazenagem (WMS)

Projeto desenvolvido para controle e organização de armazenagem, expedição e relatórios logísticos, permitindo melhor visualização de dados e apoio à tomada de decisão.

---

## 🚀 Objetivo do Projeto

O objetivo deste sistema é:
- Organizar informações de armazenagem por lotes
- Facilitar a expedição e conferência de dados
- Gerar relatórios consolidados para análise
- Melhorar a eficiência operacional do armazém

---

## 🧩 Funcionalidades

✔️ Cadastro e controle de lotes  
✔️ Organização de produtos por localização  
✔️ Relatórios separados por lote  
✔️ Exportação de dados em **CSV** e **Excel**  
✔️ Consolidação de múltiplos lotes em um único arquivo  
✔️ Estrutura pronta para ambiente de produção  

---

## 📊 Relatórios

O sistema permite:
- Exportação individual por lote  
- Exportação geral com abas separadas (Excel)  
- Arquivos CSV compatíveis com Excel, Google Sheets e Power BI  

---

## 🛠️ Tecnologias Utilizadas

- HTML / CSS / JavaScript  
- SQL  
- Git & GitHub  

> Obs: Estrutura preparada para futuras integrações com banco de dados e APIs.
